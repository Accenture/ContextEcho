Step-by-step donation guide

Follow these screens in order. The public page only starts the local wizard; the actual scan and redaction happen on 127.0.0.1, which means your own computer.

1. Run the command in Terminal

After copying the command from this page, paste it into Terminal and press Enter. The script prepares a private local environment and starts the donation wizard.

  • Leave this Terminal window open while donating.
  • When you see ContextEcho donation wizard: http://127.0.0.1:<port>/, the local wizard is running.
  • Raw sessions stay local on this machine.

2. Continue into the local wizard

Click Copy Command, paste it into Terminal, and press Enter. When the local wizard starts, this page enables Continue in Local Wizard.

  • The hosted page cannot scan your local files.
  • The local wizard runs at http://127.0.0.1:<port>.
  • Raw session history stays on your machine.

3. Discover and select a session

Click Discover Sessions. The table starts blank and fills only after a fresh local scan.

  • Choose a real session with 100+ turns or context compactions.
  • Rows marked best are usually most useful for research.
  • Click one row, then continue to Redact.

4. Redact locally and verify

Keep Full redacted for the best research fidelity, or choose User-minimized if you want stronger privacy for personal wording or feelings.

  • Add optional private words only if something specific remains.
  • Click Redact and Verify.
  • If a private word is still present, add it and run redaction again.

5. Review the verified output

A Verified clean result means the redacted file passed local checks. The automatic redaction chips summarize what was removed.

  • The redacted file is what gets submitted, not the raw source session.
  • Use the private-word checker if you want to test a specific term.
  • Check the review box, then continue to Submit.

6. Submit for maintainer review

Contributor fields are optional. Use the anonymous checkbox if you do not want your public leaderboard row to show your name.

  • Review the leaderboard preview before submitting.
  • Click Submit Donation.
  • Keep the tab open and the computer awake while upload runs.

7. Save the receipt if needed

After upload, the wizard jumps to the success result automatically. Your status is Pending maintainer review until accepted.

  • Save the submission ID if you may need support.
  • The receipt is stored locally.
  • Only verified redacted artifacts were submitted.

If automatic discovery finds nothing, keep using your coding agent and come back later. The donor flow is designed so contributors do not need to know where session history files live.
