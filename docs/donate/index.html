Donate a coding-agent session to ContextEcho

Start a local scanner, pick a real Claude Code or Codex session, redact it on your machine, and submit only the verified redacted artifact.

Raw session history stays local. The server receives only redacted files after you approve them.
One command starts the local wizard
curl -Ls https://raw.githubusercontent.com/Accenture/ContextEcho/main/scripts/run-donate.sh | bash
Privacy Details Current Ranking

Run local scannerThe command creates a private environment and opens the browser wizard automatically.

Discover sessionsThe wizard scans known Claude/Codex history folders. You do not need to know the path.
3
Redact locallyPII, paths, tokens, and custom private words are removed before upload.
4
Submit for reviewMaintainers receive the verified redacted artifact, manifest, consent, and receipt.

If automatic discovery finds nothing, the local wizard also offers a manual file picker fallback. That fallback still runs on localhost; it is not a remote raw-data upload.
